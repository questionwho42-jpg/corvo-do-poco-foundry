## Melhores práticas de CSS

- **Metodologia Consistente**: Aplique e siga a metodologia CSS consistente do projeto (Tailwind, BEM, classes utilitárias, CSS modules, etc.) em todo o projeto
- **Evite Sobrescrever Estilos do Framework**: Trabalhe com os padrões do seu framework em vez de lutar contra eles com sobrescritas excessivas
- **Mantenha o Design System**: Estabeleça e documente tokens de design (cores, espaçamento, tipografia) para consistência
- **Minimize CSS Customizado**: Aproveite utilitários e componentes do framework para reduzir a carga de manutenção de CSS customizado
- **Considerações de Desempenho**: Otimize para produção com CSS purging/tree-shaking para remover estilos não utilizados
